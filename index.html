    About

    Open Source Software, or OSS, is a small, non-profit group that prioritizes the user experience over everything else. The reason we exist is because I, Michael Goddard, got fed up of apps either being paid for, or having a bad user experience. This started when my Amazon Music application on my phone started crashing, and having several other bugs. So I decided to create my own 'little' music app.
    This app went on to be very popular. And yet, despite it's popularity, it didn't make any money. And I didn't want it too anyway. I made it to be nice to the community, and to allow people to listen to the music they want to listen to without any problems.
    I wanted to continue doing this as I enjoy making things, and i enjoy giving stuff away for free, as it makes me feel like I'm being nice. So I am creating all different sorts of software to help people and to be much more user-friendly than most other applications.
    Each project is on my personal GitHub page, and so are all Open-Sourced. If anyone wants to contribute to one of the projects, then please go to the contribute page on this website, or go to my GitHub page and choose a project you would like to contribute to. (The same thing goes for reporting errors.)
